# scikit-learn-doc-cn

sklearn库作为目前机器学习非常流行的python库，一个易读的文档更有助于工具库的理解和使用，虽然做机器学习方面的学生和工程师阅读英文并没有很大压力，但是在阅读速度上还是会有些影响。
寻找已久没找到相关的中文文档，所以想发起这个项目，欢迎大家踊跃加入！
